Автоматический браузерный агент на Python, который принимает задачу в естественном языке, интерпретирует её и выполняет действия в реальном браузере с помощью Playwright.
Проект разработан как демонстрация автономного агента с памятью, инструментами и возможностью самостоятельной навигации.

Стек технологий:
Python 3.11.9
Playwright (браузерная автоматизация)
OpenAI API

Browser_AI_agent/
agent.py                # Логика агента и принятие решений
browser_controller.py   # Управление браузером, инструменты Playwright
tools.py                # Абстракции над действиями: click, navigate, type...
memory.py               # Короткая память для предотвращения повторов
login_yandex_eda.py     # Модуль для авторизации пользователя
config.py               # Загрузка конфигурации и API ключей
main.py                 # CLI — точка входа, запуск агента
user_data/              # Папка, создаваемая проектом для локальных данных
.env_example            # Образец файла конфигурации (Для работы нужно будет вписать свой API_KEY)
.gitignore              # Игнорируется pycache, .env, временные файлы
requirements.txt        # Зависимости
README.md               # Текущее описание проекта

1. Установка проекта:
  git clone https://github.com/9BUBBLE9/Browser_AI_agent.git
  cd Browser_AI_agent

3. Установить зависимости:
  pip install -r requirements.txt

4. Установить Playwright браузеры:
   playwright install

5. Создайть или скопировать .env
   cp .env_example .env
   +заполнить:
     OPENAI_API_KEY=your_key_here
   
6. Запуск агента
   python main.py

7. После запуска появится приглашение:
   Надо сюда ввести задачу (например: 'Закажи бургер из яндекс еды на мой адрес)
   Опиши задачу для агента: ...

Как работает агент:
  LLM получает текст задачи
  Модель формирует следующий шаг
  Агент вызывает нужный инструмент:
    navigate(url)
    click_element(index)
    type_into_input(index)
    get_observation()
  Агент анализирует обновлённое состояние страницы
  Повторяет цикл до достижения результата или тупика
  Возвращает отчёт пользователю
